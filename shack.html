Phase 0 · Start now
🎯 Goal Phase 0
Get on the air as quickly as possible — before the shack is complete. With the IC-705, LiFePO4 battery and multiple portable antennas, POTA activations are already possible. FT8 on 40m at 10W is enough for 10 QSOs in 30–45 minutes.
Phase 1A · VHF Shack Core
🎯 Goal Phase 1A
Build the VHF shack station. The Yaesu FT-2980E as a dedicated 2m FM radio — always on. Diamond X-50N omni on the roof for all local repeaters. Diamond A144S5 Yagi for directional VHF, steered by the Yaesu G-450A rotator.

Phase 1B · HF Shack
🎯 Goal Phase 1B
Build the HF shack. ICOM IC-7300 MkII as the main HF transceiver. GAP Challenger DX vertical antenna permanently installed. EFOC29 coaxial cable for the antenna feed. Full HF capability from 160m to 6m.

Phase 2 · Extended VHF Station
🎯 Goal Phase 2
Expand to a full VHF/UHF station. ICOM IC-9700 for 2m, 70cm and 23cm. InnovAntennas LFA Yagi for high-gain directional 2m. Tonna 2m 11-element Yagi as a secondary antenna. Serious EME and tropo DX capability.
Phase 2
VHF/UHF/SHF Transceiver
ICOM IC-9700
Full-featured SDR-based radio for 2m, 70cm and 23cm (1.2 GHz). Direct sampling SDR architecture. Built-in spectrum scope. Supplied by HAMRADIOSHOP. The core of the extended VHF station.
Bands2m · 70cm · 23cm
Power100W (2m) · 75W (70cm) · 10W (23cm)
FeaturesSDR · Spectrum scope · D-STAR

Phase 3 · Class A — DX Station
🎯 Goal Phase 3 — Class A licence first!
The full DX station. With 100W and the Hexbeam steerable across all HF bands: crack pile-ups, rotate away from interference, eliminate QRM from behind (20 dB front-to-back). The Hexbeam looks like an umbrella — XYL-friendly. The G-450A rotator from Phase 1A is reused: no extra rotator needed.
Class A first! Phase 3
HF Beam
SP7IDX Hexbeam
6-band directional HF beam for 10/12/15/17/20/40m. ~6 dBi gain and ~20 dB front-to-back ratio. The real advantage: rotate away from rear interference. Class A licence required for 100W. G-450A rotator reused from Phase 1A.
Bands10 · 12 · 15 · 17 · 20 · 40m
Gain~6 dBi (1 S-unit improvement)
F/B Ratio~20 dB — eliminates rear QRM
